# -virtualMoneyPrediction
虛擬貨幣的分析和預測  
使用方法  
1.下載回電腦  
2.解壓縮到指定資料夾  
3.進入dist資料夾  
4.點選 main.exe  
5.等待一下  
6.點選要看的虛擬貨幣  
7.查看圖表  
圖表一個是預測 一個是實際  
#問題排除  
如果遇到問題無法執行請先看看錯誤訊息  
如果是找不到某些套件  
pip install 套件名稱  
在命令提示字元裡面執行
